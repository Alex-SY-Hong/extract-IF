# 用于提取pdf格式的论文对应的影响因子以及期刊名。

## Structure

### 期刊名的获取
提取metadata的subjects，并用正则表达式过滤出期刊名。

### IF的获取
为了更好的稳健性，直接使用Clarivate提供的影响因子文件（xlsx格式）

## 使用
单文件，随便怎么搞都行的吧┑(￣Д ￣)┍

pipreqs遇到了某种编码错误，搞不定。我在仓库里附了一个（CC-BY-4.0协议）的论文，
可供测试使用。

## 问题
不同的文件，metadata格式实际上参差不齐。比如，有文章的subjects字段是摘要，而非
期刊信息。正则表达式这些“传统”工具不很稳健，好在数量不大的情况下手动处理例外也
比较容易。

理论上AI可以改善parse的效率。比如说，把首页pdf转成图片，上传给Deepseek-OCR之类
的模型提取期刊名（如果有版权之类的问题，可以本地部署llava等）。有待验证。
